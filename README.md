## Реалізовані функції

updateAt - повертає новий список, де елемент на вказаній позиції замінено на нове значення.

updateDict - повертає нову копію словника з оновленим значенням за ключем.

append_tuple - повертає новий кортеж з доданим елементом в кінець.

incrementAge - повертає новий екземпляр з віком, збільшеним на 1.

push - імітує операцію “додавання в стек”, повертаючи нову копію списку з новим елементом на початку.

removeAt - повертає новий список без елемента за заданим індексом.
